本文是MIT课程missing-semester的学习笔记，这是一门介绍计算机工具的课，课时比较短，约为11小时，但涵盖的内容比较丰富。

Class 1: The Shell (教程地址)

shell 基础指令

  • 显示当前时间

    • 命令：date
    • 示例：
      • 输入：date
      • 输出：Sat Jan 1 00:00:00 2022

  • 打印内容到命令行

    • 命令：echo [输出内容]
    • 示例：
      • 输入：echo $HOME
      • 输出：/c/Users/MeteorVanish
      • 输入：echo Hello\ world!
      • 输出：Hello world! > 注：若参数中若含有 ' '，需用 '\ ' 代替或使用引号包裹

  • 查看程序地址

    • 命令：which [程序名]
    • 示例：
      • 输入：which echo
      • 输出：/usr/bin/echo > 注：地址分隔符在 Linux 和 macOS 下是 /，在 Windows 下是 \

  • 查看当前工作目录

    • 命令：pwd [print working directory]
    • 示例：
      • 输入：pwd
      • 输出：/c/Users/MeteorVanish

  • 切换当前工作目录

    • 命令：cd [绝对/相对路径] [change directory]
      • 绝对路径：cd /xxx
      • 相对路径：. 表示当前目录，.. 表示上级目录，- 表示之前所在目录
    • 示例：
      • 输入：cd /home
      • 作用：切换到 home
      • 输入：cd ./bin
      • 作用：切换到 home 目录下的 bin 目录
      • 输入：cd ..
      • 作用：切换到上级目录
      • 输入：cd -
      • 作用：切换到之前所在目录

  • 查看当前工作目录下文件（夹）

    • 命令：ls [ 打印目录（默认当前工作目录）] [list]
    • 示例：
      • 输入：ls （当前在 / ）
      • 输出：bin dev home ...
      • 输入：ls sys （当前在 / ）
      • 输出：block bus class dev devices firmware fs kernel module power
    • list -l
      • 权限信息：共十位
        • 第一位：d 代表文件夹， l 代表链接文件， - 代表普通文件
        • 第二至十位：三位一组，各组分别表示了文件所有者、用户组以及其他所有人的权限，每位依次为 r (read)，w (write)，x (execute)，分别代表查看、修改、执行/搜索权限，不具备相应权限则对应位置为 -
          • 对文件夹而言，r 权限表示是否能查看文件夹内包含哪些文件，w 权限表示是否能重命名、增加、删除文件夹内文件，x 权限表示是否能够进入文件夹
          • 想要访问一个文件需要具有其所有父文件夹的搜索权限

  • 重命名或移动文件（夹）

    • 命令：mv [原文件（夹）名] [新文件（夹）名]，mv [原文件（夹）地址] [新文件（夹）地址] (move)
    • 示例：
      • 输入：mv old.md new.md
      • 作用：将 old.md 重命名为 new.md
      • 输入：mv test.md ../dir2/test.md（当前在 dir1 ）
      • 作用：将 dir1 中的 test.md 移动到 dir2 中

  • 拷贝文件（夹）

    • 命令：cp [原文件（夹）名] [新文件（夹）名] [copy]
    • 示例：
      • 输入：cp test.md ../dir2/test.md（当前在 dir1 ）
      • 作用：将 dir1 中的 test.md 拷贝到 dir2 中

  • 删除文件（夹）

    • 命令：rm [文件名]，rm -r [文件夹名]，rmdir [文件夹名]（仅当文件夹为空时有效）[remove (directory)]
    • 示例：
      • 输入：rm test.md
      • 作用：删除文件 test.md
      • 输入：rm -r dir1
      • 作用：删除文件夹 dir1 及其中的所有文件
      • 输入：rmdir dir2
      • 作用：删除空文件夹 dir2

  • 创建文件夹

    • 命令：mkdir [文件夹名] [make directory]
    • 示例：
      • 输入：mkdir dir2
      • 作用：在当前目录新建名为 dir2 的文件夹

  • 查看指令用法

    • 命令：man [指令] [manual]，[指令] --help

输入输出流

  • 输入输出流重定向
    • 命令：< file，> file （覆盖rewrite），>> file (追加append)
    • 示例：
      • 输入：echo hello > hello.txt
      • 作用：在 hello.txt 文件（不存在就新建）中写入 hello
  • 显示文件内容
    • 命令：cat [文件名] [concatenate]
    • 示例：
      • 输入：cat hello.txt
      • 输出：hello
      • 输入：cat < hello.txt
      • 输出：hello
      • 输入：cat < hello.txt > hello2.txt
      • 作用：在 hello2.txt 中写入 hello.txt 的内容（在 hello2.txt 不存在时作用与 cp 相同）
  • 连接运算符|(pipe)
    • 命令：[前一条命令] | [后一条命令]
    • 功能：将前一条指令的输出作为后一条指令的输入
    • 示例：
      • 输入：ls -l / | tail -n1
      • 输出：drwxr-xr-x 1 MeteorVanish 197609 0 Jan 31 15:31 usr （ls -l / 输出内容的最后一行）
  • 将输入流重定向至输出流
    • 命令：[输入流] tee [输出流]
    • 功能：在不进入 root 终端时使用 sudo 指令修改内核变量的值
    • 示例：
      • 输入：echo 500 | sudo tee brightness
      • 作用：将屏幕亮度更改为 500 （等效于在 root 终端下 echo 500 > brightness）

杂项

  • 清空命令行快捷键：Ctrl + L

  • 进入 root 终端：sudo su （需要输入密码）

  • 退出 root 终端：exit

练习

1. 查看 shell 是否满足要求

2. 在 /tmp 下新建一个名为 missing 的文件夹

3. 用 man 查看程序 touch 的使用手册

4. 用 touch 在 missing 文件夹中新建一个叫 semester 的文件

5. 将以下内容一行一行地写入 semester 文件：

#!/bin/sh
curl --head --silent https://missing.csail.mit.edu

6. 尝试执行 semester 文件，如果程序无法执行，请使用 ls 命令来获取信息并理解其不能执行的原因

7. 通过显式地打开 sh 解释器来运行命令，将文件 semester 作为第一个参数

8. 查看 chmod 的手册

9. 使用 chmod 命令改变权限，使 ./semester 能够成功执行

10. 使用 | 和 > ，将 semester 文件输出的最后更改日期信息，写入主目录下的 last-modified.txt 的文件中

11. 写一段命令来从 /sys 中获取笔记本的电量信息
