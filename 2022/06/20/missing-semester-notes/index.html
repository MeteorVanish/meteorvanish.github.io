    本文是MIT课程missing-semester的学习笔记，这是一门介绍计算机工具的课，课时比较短，约为11小时，但涵盖的内容比较丰富。

Class 1: the shell

shell 指令

  • 显示当前时间
    • 命令： date
    • 示例输入： date
    • 示例输出： Sat Jan 1 00:00:00 2022
  • 打印内容到命令行
    • 命令： echo [输出内容]
    • 示例输入 1： echo $HOME
    • 示例输出 1： /c/Users/MeteorVanish
    • 示例输入 2： echo Hello\ world!
    • 示例输出 2： Hello world!

    注：若参数中若含有 ' ' ，需用 '\ ' 代替或使用引号包裹

  • 查看程序地址
    • 命令： which [程序名]
    • 示例输入： which echo
    • 示例输出： /usr/bin/echo

    注：地址分隔符在 Linux 和 macOS 下是 / ，在 Windows 下是 \

  • 查看当前工作目录
    • 命令： pwd (print working directory)
    • 示例输入： pwd
    • 示例输出： /c/Users/MeteorVanish
  • 切换当前工作目录
    • 命令： cd [绝对/相对路径] (change directory)
      • 绝对路径： cd /xxx
      • 相对路径： . 表示当前目录， .. 表示上级目录
    • 示例输入 1： cd /home
    • 示例输出 1：（切换到 home ）
    • 示例输入 2： cd ./bin
    • 示例输出 2：（切换到 home 目录下的 bin 目录）
    • 示例输入 3： cd ..
    • 示例输出 3：（切换到上级目录）
