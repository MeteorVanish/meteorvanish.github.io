查看当前工作目录下文件（夹）

  • 命令：ls [ 打印目录（默认当前工作目录）] <list>

  • 示例：

  • ls -l

    • 权限信息：共十位
      • 第一位：d 代表文件夹， l 代表链接文件， - 代表普通文件
      • 第二至十位：三位一组，各组分别表示了文件所有者、用户组以及其他所有人的权限，每位依次为 r (read)，w (write)，x (execute)，分别代表查看、修改、执行/搜索权限，不具备相应权限则对应位置为 -
        • 对文件夹而言，r 权限表示是否能查看文件夹内包含哪些文件，w 权限表示是否能重命名、增加、删除文件夹内文件，x 权限表示是否能够进入文件夹
        • 想要访问一个文件需要具有其所有父文件夹的搜索权限

8. 查看 chmod 的手册

10. 使用 | 和 > ，将 semester 文件输出的最后更改日期信息，写入主目录下的 last-modified.txt 的文件中

11. 写一段命令来从 /sys 中获取笔记本的电量信息

bash 中的特殊变量

  • $0：脚本名
  • $1 - $9：脚本的第一个至第九个参数
  • $@：所有参数
  • $#：参数个数
  • $?：前一个命令的返回值
    • 返回值0表示正常执行，其他所有非0的返回值都表示有错误发生。
    • true 的返回码永远是 0，false 的返回码永远是 1
    • 返回值可以搭配 &&（与运算符）和 ||（或运算符）使用，均为短路运算符
      • 示例：
  • $$：当前脚本的进程识别码
  • !!：完整的上一条命令，包括参数。
    • 常见应用：当你因为权限不足执行命令失败时，可以使用 sudo !! 再尝试一次。
  • $_：上一条命令的最后一个参数
    • 如果你正在使用的是交互式 shell，你可以通过按下 Esc 之后键入 . 来获取这个值。

通配（globbing）

shebang

  • 程序中一行以 #! 开头的代码，用于告诉 shell 如何运行程序

  • 示例：对于上方的脚本示例 example.sh，运行时只要输入 ./example.sh [arg] 即可，不用输入 shell ./example.sh [arg]

    注： 需要脚本具有执行权限，若没有则需添加权限，即 chmod +x example.sh*

  • 在 shebang 行中使用 env 命令是一种好的实践，它会利用环境变量中的程序来解析该脚本，这样就提高来您的脚本的可移植性。env 会利用 PATH 环境变量来进行定位。

  • 示例：#!/usr/bin/env python

Class 3: Editors (Vim) (教程地址)

编辑模式

  • 正常模式 [Normal]：在文件中四处移动光标进行修改
  • 插入模式 [Insert]：插入文本
  • 替换模式 [Replace]：替换文本
  • 可视化（一般/行/块）模式 [Visual (plain/line/block)]：选中文本块
  • 命令模式 [Command-line]：用于执行命令

切换快捷键

  • 按下 <ESC> 从任何其他模式返回正常模式（个人将 jk 重映射为 ESC）
  • 在正常模式，按下 i 进入插入模式， R 进入替换模式， v 进入可视（一般）模式， V 进入可视（行）模式，^V 进入可视（块）模式，: 进入命令模式

基本操作

命令行

  • :q <quit> 退出（关闭窗口）
  • :w <write> 保存（写）
  • :wq 保存然后退出
  • :e [文件名] <edit> 打开要编辑的文件
  • :ls <list> 显示打开的缓存
  • :help [指令] 打开帮助文档
    • :help :w 打开 :w 命令的帮助文档
    • :help w 打开 w 移动的帮助文档

移动

  • 基本移动：hjkl（左，下，上，右）
  • 词：w（下一个词 word），b（词初 begin），e（词尾 end）
  • 行：0（行初），^（第一个非空格字符），$（行尾）
  • 屏幕：H（屏幕首行 High），M（屏幕中间 Middle），L（屏幕底部 Low）
  • 翻页：^U（上翻 up），^D（下翻 down）
  • 文件：gg（文件头），G（文件尾）
  • 行号：[行号]G
  • 查找：f[字符]（向后查找 find），F[字符]（向前查找 Find），t[字符]（向后跳跃 to），T[字符]（向前跳跃 To）
    • 在本行内，查找光标到字符上，跳跃光标到字符前/后
    • , / ; 用于导航匹配
  • 杂项：%（找到配对，比如括号或者 /* */ 之类的注释对）
  • 搜索：/[字符串]（正向），?[字符串]（反向）
    • n / N 用于导航匹配

编辑

  • O / o 在当前行之上/之下插入空白行
  • d[移动命令] <delete> 删除 [移动命令]
    • dd 删除当前行
    • 示例：dw 删除词，d$ 删除到行尾，d0 删除到行头
  • c[移动命令] <change> 改变 [移动命令]
    • 作用等同于 d[移动命令] 后 i
    • cc 删除当前行并进入插入模式
    • 示例：cw 改变词，c$ 改变到行尾，c0 改变到行头
  • x 删除字符（等同于 dl ）
  • s 删除替换字符（等同于 xi ）
  • r <replace> 直接替换字符
  • u <undo> 撤销，^R <redo> 重做
  • y <yank> 复制，p <paste> 粘贴
  • ~ 改变字符的大小写
  • . 重复刚才的输入内容

计数

用一个计数来结合“名词”和“动词”会执行指定操作若干次。 - 示例： - 3w 向前移动三个词 - 5j 向下移动5行 - 7dw 删除7个词

修饰语

  • i <inside> 表示“内部”，a <around> 表示“周围”
  • 示例：
    • ci( 改变当前括号内的内容
    • ci[ 改变当前方括号内的内容
    • da' 删除一个单引号字符串，包括周围的单引号
